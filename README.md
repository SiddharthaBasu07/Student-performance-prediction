## 📋 Project Overview
A machine learning web application that predicts student academic performance based on demographic, academic, and behavioral factors. The system helps educational institutions identify students who may need additional support.

## ✨ Features
- 🔍 **Single Student Prediction**: Input 30+ features to predict pass/fail
- 📊 **Model Comparison**: Compare Decision Tree, Random Forest, and Gradient Boosting
- 📈 **Feature Analysis**: Understand which factors most influence performance
- 🎨 **Interactive UI**: Beautiful, responsive Streamlit interface
